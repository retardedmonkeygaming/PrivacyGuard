# Privacy Guard

A safe, transparent tool to remove Microsoft bloatware and control Windows telemetry.


## -> Features
- **40+ Microsoft bloatware apps** - Remove unwanted pre-installed apps
- **Delete All option** - One-click removal of all bloatware
- **Granular selection** - Choose specific apps to remove
- **Advanced telemetry control** - Disable diagnostic data, ads, location tracking
- **Full restore capability** - Reinstall all removed apps with one click
- **Safety first design** - Never removes critical system components
- **Detailed logging** - All operations logged to Desktop

## -> Microsoft Bloatware List
- **Core Apps**: OneDrive, Xbox, Candy Crush, Skype, Photos, Calculator
- **Productivity**: Teams, Outlook Mail, Calendar, Sticky Notes, To Do
- **Entertainment**: Minecraft, Solitaire, Paint 3D, Mixed Reality Portal
- **Utilities**: Feedback Hub, Tips, People, Phone Link, Your Phone
- **Additional**: Bing News, Weather, Finance, Sports, Health, Money, Translator

## -> Safety Notes
- **This program never removes critical system apps** (ShellExperienceHost, Cortana, etc.)
- **All operations require explicit confirmation** (`DELETE ALL` for bulk removal)
- **Requires Administrator rights** for full functionality
- **100% reversible** - Restore all apps anytime
- **Zero telemetry** - This tool collects no data

## -> Usage
1. Download `PrivacyGuard.exe` from [Releases](https://github.com/YOUR-USERNAME/PrivacyGuard/releases)
2. **Right-click → "Run as administrator"**
3. Select apps to remove or choose "Delete ALL Microsoft Bloatware"
4. Type confirmation word when prompted
5. Check `privacy_guard_log.txt` on Desktop for results

> -> **First run warning**: Windows SmartScreen may show "Protected your PC"  
> Click **"More info" → "Run anyway"**

## -> Requirements
- Windows 10 version 1607+ or Windows 11
- .NET Framework 4.8+
- Administrator privileges

## -> Privacy Promise
This tool:
